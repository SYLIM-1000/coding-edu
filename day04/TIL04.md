## CSS 적용 방법
1. 코드 내부에 style 코드를 넣는 방법
2. head 코드 내에 style 코드를 넣는 방법 (내부 스타일)
3. 동일한 폴더 내에 style.css 파일을 만들고 해당 파일에 디자인 양식을 저장 (외부 스타일)

## 선택자
- Id: 고유한 친구라서 문서에서 한 번만 사용할 수 있음
- Class : 한 문서에서 여러 번 사용할 수 있다

### 자식/후손 선택자
- 자식 선택자
    바로 아래에 있는 자식 요소
- 후손 선택자
    특정 요소 안에 들어있는 모든 요소
    만약 nav > ul > li > a 4가지 태그가 있다면, 
    nav의 자식 선택자는 ul
    nav의 후손 선택자는 ul, li, a 모두
- 자식/후손 구분 방법
    nav > lu > li {
        color: pink;
    } -> 이 코드는 자식 선택자 (nav 바로 아래 자식 lu만 바뀜)
    nav lu {
        color: red; 
    } -> 이 코드는 자손 선택자 (nav아래 있는 모든 lu가 바뀜)

### 속성 선택자
    <input type="text"> 코드의 특정 속성에도 CSS를 적용할 수 있는데 아래와 같음
    input[type="text"] {
        border: 2px solid blue;
    }  --> 해석하자면, 인풋 타입의 형태에 따라 CSS를 적용할 수 있음
