01. Core Syntax

HTML 테이블을 구성하는 4가지 핵심 태그의 이해

테이블 구조의 핵심 요소

<table>

표 전체를 감싸는 가장 바깥쪽 컨테이너입니다. 모든 데이터는 이 안에 위치합니다.

<tr>

Table Row의 약자로, 표의 가로 줄(행)을 정의할 때 사용합니다.

<th> / <td>

th는 제목 셀(굵게), td는 일반 데이터 셀을 의미합니다.

3x3 실무 데이터 테이블 예시

이름 나이 사는곳
홍길동 25 서울시 강남구
김철수 30 부산시 해운대구
이영희 28 광주시 북구

* 위 테이블은 th와 td를 적절히 배합하여 가독성을 극대화한 구조입니다.

02. Styling & Ethics

더 깔끔하고 전문적인 표를 만드는 디자인 기법

전문가급 테이블 스타일링 팁

Border Collapse

border-collapse: collapse; 속성을 사용하여 셀 사이의 이중 테두리를 하나로 합치면 훨씬 세련된 느낌을 줍니다.

Cell Padding

텍스트와 테두리 사이에 충분한 Padding(여백)을 주어 데이터가 숨을 쉴 수 있는 공간을 확보하세요.

의미론적(Semantic) 태그 활용

  • <thead>: 제목 행들을 그룹화하여 스크롤 시 고정하거나 별도 스타일링이 가능합니다.
  • <tbody>: 본문 데이터 영역을 정의하여 코드의 가독성을 높입니다.
  • <tfoot>: 합계나 요약 정보를 표 하단에 표시할 때 사용합니다.

테이블 데이터의 시각적 변환 (나이 비교)

표 형식의 데이터를 위와 같이 차트로 시각화하면 비교 분석이 더욱 용이해집니다.

데이터 무결성 및 가독성

100%
가독성 향상 수치

구조화된 테이블은 일반 텍스트 나열보다 정보 전달 속도가 약 2배 이상 빠르며, 복잡한 비즈니스 데이터의 오류를 방지합니다.

실무에서의 테이블 활용 사례

  • 1. 재무 및 통계 리포트
    방대한 수치 데이터를 일목요연하게 정리합니다.
  • 2. 관리자 대시보드
    사용자 목록, 주문 내역 등을 효율적으로 노출합니다.
  • 3. 비교 분석 표
    제품 사양이나 서비스 플랜 간의 차이를 시각화합니다.
