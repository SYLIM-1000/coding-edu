## 🎯 과제 개요
HTML, CSS, Bootstrap, Git까지 지금까지 학습한 모든 내용을 활용하여 **Day1에 제작했던 자기소개 페이지를 새롭게 리뉴얼**합니다.
Day1에서는 HTML을 이용해 웹페이지의 구조를 만드는 것에 집중했다면, 이번 프로젝트에서는 CSS를 활용한 디자인, Bootstrap을 이용한 반응형 레이아웃, Git을 활용한 버전 관리까지 적용하여 하나의 완성된 웹페이지를 제작하는 것이 목표입니다.
이번 미니프로젝트는 지금까지의 학습 내용을 정리하는 과정이며, 앞으로 JavaScript와 React를 배우기 전 자신의 실력을 점검하는 첫 번째 결과물입니다.

---

# 💻 기본 과제 요구사항
### 필수 구현
#### 1. 자기소개 페이지 제작
Day1에 만들었던 자기소개 페이지를 새롭게 제작합니다.
아래 내용을 포함합니다.
- 자기소개
- 프로필 이미지
- 취미 또는 관심사
- 목표 또는 각오

---

#### 2. 시맨틱 태그 활용
아래 태그를 적절하게 사용합니다.
- header
- nav
- main
- section
- article
- footer

---

#### 3. CSS 스타일 적용
직접 작성한 CSS를 활용하여
- 색상
- 폰트
- 여백
- 카드 UI
- Hover 효과
등을 적용합니다.

---

#### 4. Bootstrap 활용
Bootstrap 컴포넌트를 3개 이상 활용합니다.
예시
- Navbar
- Card
- Button
- Grid
- Container
- Alert
- Badge

---

#### 5. 반응형 웹 구현
Bootstrap Grid 또는 Responsive Utility를 활용하여
Desktop, Tablet, Mobile 환경에서 자연스럽게 표시되도록 구성합니다.

---

#### 6. Git 활용
프로젝트를 Git으로 관리합니다.
- Commit 3회 이상
- GitHub Repository에 Push

---

# ⚠️ 주의사항
- HTML 시맨틱 구조를 고려하여 작성할 것
- Bootstrap만 사용하는 것이 아니라 CSS도 함께 활용할 것
- 반응형 화면을 Desktop, Tablet, Mobile에서 모두 확인할 것
